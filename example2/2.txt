# Функции в JavaScript

В JavaScript функции - это "объекты первого рода", их можно хранить в переменных, передавать в качестве аргументов в другие функции и т.п.

Пример
	var f = function(x) {return x + 1};
Переменной а присвоена функция. Синтаксис функции такой же как обычно, но не пишется имя (В S6 f = x => x + 1;)

Если функция задана традиционным способом, ее все равно можно использовать как объект первого рода:
	function inc(x) {
		return x + 1;
	}
	
	var f = inc; // т.е. inc это имя переменной, содержащей функцию
В случае задания функции традиционным способом ей можно пользоваться до того, как она описана

	function add(x, y) {
		return x + y;
	}
	
	// функция strange получает функцию двух аргументов и возвращает ф-ю одного аргумента, фиксируя второй
	function strange(f, a) {
		return function(x) {
			return f(x, a);
		}
	}
	
	var inc = strange(add, 1);
	concole.log(inc(10));

## функция setInterval и setTimeout

Эти функции встроены в JavaScript
setInterval(функция, время в милисекундах). После этого функция начинает вызываться с указанной периодичностью

setTimeout - аналогично, но выполнить один раз.

requestAnimationFrame() - аналогично setInterval, но не нужно казывать время, действие повторяется (обычно) 60 раз в секунду в тот момент, когда браузеру удобно что-нибудь нарисовать.
